# Screenshots Guide

Place your screenshots in this folder with the following names:

## Required Screenshots:

1. **01-workspace-created.png**
   - Workspace successfully created in Terraform Cloud
   - Shows workspace name and configuration

2. **02-aws-credentials.png**
   - AWS Access Key and Secret Key added as environment variables
   - Shows both variables marked as "Sensitive"

3. **03-dev-env-variables.png**
   - Dev workspace variables configured
   - Shows environment variables and Terraform variables

4. **04-test-env-variables.png**
   - Test workspace variables configured
   - Shows environment variables and Terraform variables

5. **05-all-workspaces.png**
   - All three workspaces visible (Dev, Test, Prod)
   - Shows workspace list in Terraform Cloud

6. **06-email-notification-dev.png**
   - Email notification configuration for Dev environment
   - Shows notification settings and triggers

7. **07-email-verification.png**
   - Email notification verification received
   - Shows confirmation email from Terraform Cloud

8. **08-plan-pending.png**
   - Terraform plan stuck in queue
   - Shows "Waiting for resources" message

## How to Add Screenshots:

1. Take screenshots from your Terraform Cloud account
2. Save them with the exact names listed above
3. Place them in this `screenshots/` folder
4. The documentation will automatically reference them

## Screenshot Tips:

- Use PNG format for better quality
- Capture the entire relevant section
- Make sure text is readable
- Hide any sensitive information (if not already masked)
- Use consistent browser window size
